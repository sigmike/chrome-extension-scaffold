# Chrome extension scaffold

Generate a scaffold for a Chrome extension.

By default it generates HAML and CoffeeScript files.

## Installation

Or install with:

    $ gem install chrome_extension_scaffold

## Usage

    $ chrome_extension_scaffold my_extension_name
